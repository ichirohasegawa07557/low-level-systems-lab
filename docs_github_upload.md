# GitHub Upload Guide

Recommended repository name:

```text
low-level-systems-lab
```

Recommended description:

```text
Portable C systems lab: custom CPU assembler/emulator, numerical kernels, and tiny x86 OS source with optional boot image generation.
```

Upload commands:

```bash
git init
git add .
git commit -m "Initial portable low-level systems lab"
git branch -M main
git remote add origin https://github.com/ichirohasegawa07557/low-level-systems-lab.git
git push -u origin main
```
