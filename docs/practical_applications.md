# Practical Applications

This repository is useful as a portfolio and learning platform for several fields.

## Backend and Infrastructure Engineering

The project demonstrates careful C implementation, build automation, binary formats, and CI testing.

## Data Engineering and ML Infrastructure

The matrix kernel connects directly to ML infrastructure concepts such as vector operations, inference kernels, and feature computation.

## Robotics and Embedded Systems

The CPU emulator and tiny OS modules provide a path toward embedded control, firmware concepts, memory-mapped I/O, and deterministic execution.

## Security and Reverse Engineering

The disassembler, bytecode format, memory model, and bootloader are useful foundations for reverse engineering and binary analysis.

## Research Portfolio

The repository shows that the author can move below application-level programming and reason about execution, memory, numerical kernels, and OS bootstrapping.
